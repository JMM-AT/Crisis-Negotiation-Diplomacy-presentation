# Crisis Negotiation & Diplomacy Bureau — 3D Models Presentation

An interactive 3D visualization showcasing the Crisis Negotiation & Diplomacy Bureau through animated animal models.

## Overview

This project presents a 3D interactive experience featuring various animal models that represent different aspects of crisis negotiation and diplomacy. Each model is displayed in a sleek, immersive interface with detailed information panels.

## Features

- **3D Model Gallery**: Interactive 3D models of various animals including eagle, elephant, dolphin, octopus, owl, honeybee, and crow
- **Immersive UI**: Dark theme with gold accents, featuring a top bar, information panel, and member navigation
- **Loading Screen**: Animated loading sequence with progress indicators
- **Responsive Design**: Optimized for modern web browsers

## Models Included

- Eagle (🦅)
- Elephant
- Dolphin
- Octopus
- Owl
- Honeybee
- Crow

## How to View

Simply open `bureau-glb.html` in a modern web browser that supports WebGL and 3D rendering.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Three.js for 3D rendering)
- GLB 3D models

## Browser Requirements

- Modern browser with WebGL support
- Recommended: Chrome, Firefox, Safari, or Edge
